masmvaliweb
===========
Web Service for [MASMVALI](https://github.com/inodb/masmvali), Metagenomic
Assembly VALIdation.

Install
============
```
git clone http://github.com/inodb/masmvaliweb
cd masmvaliweb
pip install -r requirements.txt
```

Run development server
======================
```
python app.py
```
